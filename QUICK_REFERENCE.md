## 🚀 Advanced Features

### 🎤 Voice Input
**How to use:**
1. Click microphone icon (🎤)
2. Allow microphone access
3. Speak your issue
4. Text appears automatically

**Supported:** Chrome, Edge

### ✨ AI Suggestions
**How it works:**
1. Type issue description
2. AI detects keywords
3. Suggestions appear
4. Click to add tags

**Keywords:** pothole, garbage, streetlight, water, drainage, park, traffic, noise

### 🌙 Dark Mode
**How to enable:**
1. Go to Settings (⚙️)
2. Find "Dark Mode"
3. Toggle switch
4. Saved automatically

### 🔔 Notifications
**Features:**
- Real-time updates
- Filter by type
- Mark all read
- Delete individual
- Time stamps

## 🔧 Quick Actions

### Login
- Quick Access: Click "Citizen" or "Admin"
- Or create account via signup

### Report Issue
1. Fill name & issue
2. Click "Detect Location"
3. Upload photo (optional)
4. Submit

### Search Location (Map)
- Type: "Patna Junction"
- Or: "Gandhi Maidan, Patna"
- Press Enter

### Update Status (Admin)
1. Find complaint
2. Click "Start Work" or "Mark Resolved"
3. Status updates

### Enable Dark Mode
1. Settings → Appearance
2. Toggle "Dark Mode"
3. Done!

## 🎯 Tips & Tricks

### Better Location Search
- Add city/state: "Patna Junction, Bihar"
- Use landmarks: "Gandhi Maidan"
- Check spelling

### Voice Input
- Speak clearly
- Pause before stopping
- Check microphone permissions

### AI Suggestions
- Type full sentences
- Include keywords
- Click suggestions to add

### Dark Mode
- Reduces eye strain
- Saves battery (OLED)
- Looks professional

## 🐛 Troubleshooting

### Voice input not working?
- Check browser (Chrome/Edge)
- Allow microphone access
- Check system permissions

### Location search fails?
- Add more details
- Include city/state
- Check internet connection

### Dark mode not saving?
- Check browser storage
- Clear cache
- Try again

### Notifications not showing?
- Refresh page
- Check filters
- Enable notifications in settings
